OpenGL Project

This is my 2nd year Graphics project where I learnt how to create an OpenGL game from scratch. I aimed to make an flying game in space where you endlessly fly through gates, inspired by Superman 64. The controls are first person with floating-like movement, as if you are moving in space. Once you collide with a gate, the next one is created and you will gain a movement speed boost. Several asteroids are scattered within a varying distance from the planet. The asteroids orbit around the planet.
